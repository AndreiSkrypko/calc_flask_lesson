# Готовый калькулятор на Flask (эталон)

Эталонный проект для **Урока 1**. Используй его, чтобы:

- проверить, как должен работать калькулятор;
- сравнить свой код с готовым;
- запустить пример: `pip install -r requirements.txt` → `python app.py` → http://localhost:5000

Создавай свой проект в PyCharm по шагам интерактивного руководства в папке `lesson1`.
